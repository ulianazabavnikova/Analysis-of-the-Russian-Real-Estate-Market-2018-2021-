# Analysis-of-the-Russian-Real-Estate-Market-2018-2021-
В рамках учебного проекта проведен комплексный анализ рынка недвижимости РФ на основе данных объявлений о продаже жилой недвижимости за период 2018–2021 гг. (более 5 млн записей). Основное внимание уделено исследованию цены за квадратный метр, выявлению факторов, влияющих на стоимость объектов, а также построению прогнозной модели.

realty_18_21.csv: 5,4 млн объявлений о продаже жилья в РФ (2018–2021)

ссылка для скачивания: https://drive.google.com/file/d/1qsyhyHYmW6NajQQ4OyKmlhyDSo-87V-9/view?usp=sharing

region.csv: cправочник регионов

realty_k3_3.ipynb: ноутбук с анализом (полный цикл обработки данных, визуализация, выявление факторов ценообразования и построение прогнозной модели)
